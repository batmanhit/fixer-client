# FIXER Test Frontend

Apollo Server with Express.

[Demo](http://fixertest.s3-website-us-west-2.amazonaws.com/)

## Installation

* `yarn install`
* `yarn start`
* visit `http://localhost:3000` for Fixer frontend

#### Testing

* `yarn test`

- Frontend: 10 hours
- Backend: 5 hours
